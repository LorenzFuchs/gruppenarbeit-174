How does Add Task work

Join is a Kanban management program with which tasks are added in the Add Task section. These tasks get certain parameters such as title and description. They are also assigned to a category that determines their urgency.

How does Backlog work

In Backlog, the data stored in Add Task is displayed. Here you can edit and delete any task.

How does Board work

In Board, the tasks from the backlog are displayed in their processing status. This includes the first status "To Do" and the last status "Done". You can move the tasks to another status using drag&drop.
